Known bugs and limitations

  • Using old version model files. Existing results related to FE parts and beams created in older versions of FEDEM are not recognized when the model is opened in version 7.0 and later. This is due to the change of "Link" to "Part" or "Beam" as type names of these objects. The old FRS-files use the "Link" keyword which is no longer recognized. The solution is to delete all results, including the reduction if component modes have been used, and do a new simulation.
  • Memory overflow for large models. When running an animation over a long time series for a large model, the memory my be exhausted. Resolve by closing the animation window after a solving process is started, or toggle off "Automatically start a simultaneous Rigid Body animation on solver start" in the Output tab of the Advanced Solver Setup dialog.
  • Exporting animation aborts the GUI. When exporting an animation to MPEG or AVI, the user interface shuts down after the export is finished.
  • Graph view and properties. The sub-title field is now absent.
  • Number of curve symbols. It is not possible to select how many symbols to use on each curve. There will always be one symbol for each data point, or no symbols at all.
  • Zoom with auto-scale. The button and menu entry for zooming a graph view with auto-scaling to the windowed curve points is no longer available.
  • Print graph view. The button and menu entry for printing a graph view is no longer available.
